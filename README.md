# 45-Second Timer

Простое веб-приложение таймера обратного отсчета на 45 секунд с визуальной индикацией и звуковым сигналом.

## Возможности

- **Таймер на 45 секунд**: Циклический отсчет времени.
- **Звуковой сигнал**: Воспроизведение короткого сигнала (бип) в конце каждого интервала.
- **Визуализация**: Круговой индикатор прогресса, меняющийся по мере истечения времени.
- **Управление**: Кнопки Старт/Стоп для управления таймером.

## Технологии

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Установка и запуск

1. **Установите зависимости:**

   ```bash
   npm install
   ```

2. **Запустите режим разработки:**

   ```bash
   npm run dev
   ```

   Приложение будет доступно по адресу `http://localhost:5173` (или другом порту, указанном в консоли).

## Сборка для продакшена

```bash
npm run build
```
